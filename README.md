# Devs.ai App Builder - Vite + React + SQLite

This is a [Vite](https://vite.dev) + React project with SQLite support, designed for the Devs.ai App Builder.

## Getting Started

First, install dependencies:

```bash
pnpm install
```

Then run the development server:

```bash
pnpm run dev
```

Open [http://localhost:3005](http://localhost:3005) with your browser to see the result.

You can start editing the page by modifying `src/App.tsx`. The page auto-updates as you edit the file.

## SQLite GUI

To view and manage your SQLite database, run:

```bash
pnpm run sqlite-gui
```

This will start a GUI at [http://localhost:8080](http://localhost:8080).

## Build

To build for production:

```bash
pnpm run build
```

Preview the production build:

```bash
pnpm run preview
```
